## Setup Instructions

1. **Add Your Images**: Place your phone mockup images in the `public/` folder:
   - `hero-phone.png` - Main hero section phone mockup
   - `benefits-phone.png` - Benefits section phone mockup

2. **Open in Browser**: Simply open `index.html` in your web browser

3. **Customize**: Edit the HTML, CSS, or JavaScript files to match your branding

## Sections

1. **Navigation** - Sticky header with smooth scroll links
2. **Hero Section** - Eye-catching hero with animated phone mockup and floating cards
3. **Features** - 4 main feature cards with icons
4. **Benefits** - Detailed benefits list with image
5. **Additional Features** - 4 additional feature showcase cards
6. **CTA Section** - Call-to-action with gradient background
7. **Getting Started** - 4-step process cards
8. **Footer** - Complete footer with links and social media

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary: #6366F1;
    --secondary: #EC4899;
    --accent: #06B6D4;
    /* ... more colors */
}
```

### Content
Edit the text directly in `index.html`

### Animations
Adjust animation timings in `styles.css` and `script.js`
