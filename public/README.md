# Images Folder

Place your REMPIC app images here.

## Required Images:

1. **hero-phone.png** 
   - Main hero section phone mockup
   - Recommended size: 600x1200px or similar aspect ratio
   - Should show the REMPIC app interface with cards/transactions

2. **benefits-phone.png**
   - Benefits section phone mockup
   - Recommended size: 600x1200px or similar aspect ratio
   - Should show account management or financial features

## Image Tips:

- Use PNG format with transparent background for phone mockups
- High resolution images (at least 2x for retina displays)
- Optimize images for web (compress without losing quality)
- Keep file sizes reasonable (<500KB per image)
